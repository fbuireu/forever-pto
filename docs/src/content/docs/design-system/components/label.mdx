Source: `src/ui/modules/core/primitives/Label.tsx`. A styled native `<label>` (small, medium weight, tight leading). It carries no logic — association happens through the standard `htmlFor` attribute passed via props.

<LabelDemo client:visible />

## Usage

```tsx
import { Input } from '@ui/modules/core/primitives/Input';
import { Label } from '@ui/modules/core/primitives/Label';

<Label htmlFor='pto-days'>PTO days per year</Label>
<Input id='pto-days' type='number' />
```

## Conventions

- `peer-disabled:` styles dim the label and switch the cursor when a sibling control marked `peer` is disabled. The control must precede the label in the DOM for Tailwind's peer selector to apply — reorder visually with CSS (`order-*`) if the label should appear first.
- The app's form layer (`src/ui/modules/core/primitives/Form.tsx`, reference-only here) wraps this same component as `FormLabel` and adds error coloring plus automatic `htmlFor` wiring.
- Pair with [Input](/design-system/components/input/) and [Textarea](/design-system/components/textarea/).
